# Bank Gateway - Capabilities

## 1. Domain Purpose
The Bank Gateway is the "Outer Ring" for all monetary movements. It handles the communication with financial institutions and normalizes various banking formats.

## 2. Core Capabilities
*   **Bankgirot Integration:** Reading and parsing CAMT/PAIN files for automated reconciliation.
*   **Real-time Balance Ingestion:** Fetching current cash positions via Open Banking APIs.
*   **Payment Execution:** Securely sending outbound payment orders to Swedbank.
