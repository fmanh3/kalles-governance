# Invoicing Gateway - Capabilities

## 1. Domain Purpose
The Invoicing Gateway acts as the "Outer Ring" for incoming vendor invoices. It handles document ingestion, OCR, and validation before handing over to the AP Engine.

## 2. Core Capabilities
*   **Multi-channel Ingestion:** Receiving invoices via Email, Portal, or Peppol.
*   **OCR & Data Extraction:** Automatically extracting amounts, OCR numbers, and dates.
*   **Duplicate Detection:** Preventing double payments of the same invoice number.
