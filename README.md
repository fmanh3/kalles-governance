# Kalles Buss AB - Company as Code (Governance)
Detta repository utgör bolagets samlade regelverk och styrdokument. All text här fungerar som kravspecifikation för våra AI-agenter och underliggande IT-system (Event-Driven DDD på GCP). Ändringar här triggar automatiskt vår CEO-Agent (GitHub Actions).

## Projektets Repositories
Här är de fem huvudkomponenterna som utgör systemet:

1.  [**kalles-governance**](https://github.com/fmanh3/kalles-governance) - Strategisk styrning, ADR:er och IaC (Terraform).
2.  [**kalles-finance**](https://github.com/fmanh3/kalles-finance) - Huvudbok, fakturering och ekonomisk compliance.
3.  [**kalles-hr**](https://github.com/fmanh3/kalles-hr) - Personalplanering och regelefterlevnad för förare.
4.  [**kalles-traffic**](https://github.com/fmanh3/kalles-traffic) - Fleet management och trafik-simulator.
5.  [**kalles-customer-success**](https://github.com/fmanh3/kalles-customer-success) - VD-portal och förarapplikation (Frontend/BFF).
