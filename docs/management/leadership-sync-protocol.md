# Ledningsgruppens Arbetsordning (Governance)

## 1. Syfte
Att säkerställa synkronisering mellan bolagets autonoma domäner och ge CEO (människa) en samlad lägesbild för strategiska beslut.

## 2. Roller i Ledningsgruppen
Varje roll representerar en samling underliggande agenter:

| Roll | Ansvarig Domän | Representerar Agenter |
| :--- | :--- | :--- |
| **CFO Agent** | Finance | Treasury, Tax, AP, Billing, Energy Strategist, Controlling |
| **Traffic Manager** | Traffic | Scheduling Agent |
| **HR Manager** | HR | Rostering Agent |
| **Fleet & Depot Manager** | Depot | Maintenance, Charger Agent |

## 3. Rapporteringsstruktur (Daily Sync)
Varje dag kl 06:00 genereras en rapport med följande struktur:
1. **Status:** Grön/Gul/Röd status för domänen.
2. **Kritiska Händelser:** Vad har hänt senaste 24h.
3. **Beroenden:** Vilken hjälp behövs från andra domäner.
4. **CEO Decision:** Specifika punkter där mänsklig intervention krävs.
